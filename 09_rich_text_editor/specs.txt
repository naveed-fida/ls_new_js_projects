A rich-tex editor

The text editor should allow it's users to:
  - Type text into it
  - make text bold
  - italicize text
  - underline text
  - strikethrough text
  - create a link from text
  - create an ordered list
  - create an unordered list
  - align text to right, left, center and justify

You will find the following references useful as you work on this project:

  1. https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes/contenteditable
  2. https://developer.mozilla.org/en-US/docs/Web/API/Document/execCommand
