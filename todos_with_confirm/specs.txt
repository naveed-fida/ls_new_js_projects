A list of todos each with custom prompt for deletion

- Create a Dom list from the given collection of todos
- Add a delete button for each todo


- When the delete button is clicked a prompt appears asking the user to confirm.

- The prompt:
  - is a custom, css based prompt
  - has a yes and a no button
  - when the user clicks yes, the prompt disappears and corresponding todo is
    deleted

  - if the user clicks no, the promp disappears and the todo is not deleted
